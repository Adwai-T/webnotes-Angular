Heroku

Create a new application

  • heroku create : Create app with random name
  • heroku create appName : Create app with custom name
  • heroku apps:rename : Rename an application.

Deploy app to heroku.

  • git push heroku master : The application should be create and a git commit needs to be done before that as the last commit will be pushed to heroku.

Setup Enviroment Variables

  • heroku config:set key=value : Set a Enviroment valiable.
  • heroku config : Get a list of all the set configuration values.
  • heroku config:unset key : Remove a key from the configuration/Environment set

Open Application

    heroku open : After the application has been deployed we can open the application by using this.

View Logs

    heroku logs --tail

Fixing App not found

Some times you might have an issue when you try to push your git-commit to heroku. The error says there was no app found. Following are steps to fix this problem.
  • git remote -v : Check for the current url
  • git remote rm heroku : remove remote url
  • git remote add heroku git@heroku.com:boiling-inlet-6957.git : re-add the remote url

Dyno Management

  • heroku ps : List of dynos for an application
  • heroku ps:restart : Restart all dynos
  • heroku ps:restart web : Adding a name to the above will restart that dyno
  • heroku logs --tail : View Logs
  • heroku ps --help : Get help
